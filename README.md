# Simple Text to speech App 
This is a basic Text-to-speech web application built using HTML,CSS and JavaScript..

## Features
- Convert text into speech.
-Simple and clean user interface.
One-click"speak" button.

## Technologies Used 
-HTML
-CSS
-JavaScript(SpeechSynthesis -API)

## How to use 
1.Enter text in the input field .
2. Click on the "Speak" button.
3. The browser will read the text  aloud .
